Hoe werkt het?

  1. Login met GitHub - Dit geeft de app toegang om webhooks te registreren
  2. Dien je repository in - Kies een opdracht en vul je GitHub repo URL in
  3. Webhook wordt automatisch aangemaakt - De app registreert een webhook op je repo
  4. Push je code - Werk aan je opdracht, commit en push
  5. AI analyseert automatisch - Bij elke push wordt je code geanalyseerd
  6. Bekijk feedback - Klik op "Feedback" om de AI suggesties te zien
Vereisten:
  • Je moet eigenaar zijn van de repository (of admin rechten hebben)
  • De repo mag publiek of privé zijn
  • Je GitHub account moet de app geautoriseerd hebben
